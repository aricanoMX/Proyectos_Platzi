PHP es un lenguaje de programación de propósito general de alto nivel que fue diseñado especialmente para el desarrollo de aplicaciones web.

Es interpretado, multiplataforma, open source, el cual ha sido muy popular en los últimos años.

¿Qué NO es PHP?

No es un lenguaje compilado, por lo cual siempre tendrás que llevar juntos tu código y tu interprete.
No esta diseñado para realizar aplicaciones de escritorio.
Para trabajar con PHP instalaremos un entorno de desarrollo llamado XAMPP, no es un entorno pensado para producción.

XAMPP es un entorno de producción solamente para que trabajemos en nuestra etapa de desarrollo.

X = cualquier sistema operativo
A = apache
M = mariadb, similar a MYSQL casi con las mismas caaracteristicas.
P = php
P = perl

------------------------------ Variables

para inicializar una variable comenzamos con el signos de $ seguido del nombre de nuestra variable.
Un nombre valido en PHP comienza con "_", o con una letra.


$_ejemplo = 1 

si inicializamos con comillas y una cadena de caracteres se puede decir que este es un string.

$name = 'Andrés Ricaño César';

y para llamarla se puede llamar $name, con el nombre de la variable

php no tiene un sistema de tipos estatico 
es decir que cuando se declara una variable esta no tiene un tipo definido.

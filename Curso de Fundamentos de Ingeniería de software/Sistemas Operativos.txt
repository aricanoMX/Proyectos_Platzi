Qué es software libre y que es Open Source?
    Libre implica que cualquier persona puede reescribir y modificar el sistema operativo.
        
    Open Source es codigo abierto que se puede redistribuir, pero no garantiza la libertad del codigo.

    BSD Licence determina cómo se puede distribuir el código.
    GNU / GPL Las personas que redistribuyen el código tienen que aportar.
    Kernel es el núcleo del Sistema Operativo y corre en el anillo 0.
    UNIX el primer sistema operativo.

    XEROX Star - Xerox creo la computación movil, mouse, sistemas operativo que creó las ventanas.

-------------------- Permisos, niveles de procesos y provolegios de ejecución --------------------
En sistemas UNIX se pueden artibuir permisos a distintos grupos:

Permiso:
    en sistemas unix con Chmod se pueden modificar los permisos a los que se puede tener acceso.
    read(r): permiso para leer archivos.
    write(w): permiso de escritura, ejemplo  r w x, r w -, r - -
    Execute(X): permiso para ejecutar.

el comando sudo eleva los permisos que tiene a nivel administrador.

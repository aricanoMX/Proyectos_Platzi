Qué es fronten y que es backend?

Frontend 
    Parte del software que interactua con el usuario.

    Lenguajes
        estos son los lenguajes que interactuan con el usuario

        HTML: es la estructura de la pag web.
        CSS: da estilos al documento HTML.
        JS: lleva toda la parte funcional de la web.

        Compilador: facilita la creación de paginas web.
